# Trot — Landing UI kit

A static recreation of the trot.dog single-page landing site. Three sections plus footer, exactly per `docs/landing.md`:

1. **Hero** — wordmark + headline + subhead + email-capture CTA + phone screenshot of Home
2. **Three feature blocks** — Walks log themselves, Targets that fit your dog, A streak worth keeping
3. **Secondary CTA** — same email capture, in the deep evergreen secondary colour
4. **Footer** — Privacy, Terms, X, copyright

Plain HTML/CSS, no framework, no build step — exactly how the brief specifies the real site should ship. Drops directly into `web/` once final.
